# SpaceXplore

This is a simple app that allows you to explore the different SpaceX rockets and dragons via the [SpaceX API](https://github.com/r-spacex/SpaceX-API).

This project also uses [Ant Design](https://ant.design/) as the framework for some of the components.

## To run the project

In the project directory, you can run:

### `yarn start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## To test the project

### `yarn test`

Launches the test runner in the interactive watch mode.

## To build the project

### `yarn build`

Builds the app for production to the `build` folder.<br />
It correctly bundles React in production mode and optimizes the build for the best performance.

## Other info
